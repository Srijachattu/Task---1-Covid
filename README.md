# COVID-19 Data Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the COVID-19 dataset using Python. It analyzes worldwide COVID-19 trends, including confirmed cases, deaths, recoveries, and active cases, and visualizes the results using various charts.

---

## 📂 Dataset

**Dataset Used:** `full_grouped.csv`

The dataset contains daily COVID-19 statistics for countries around the world, including:

- Date
- Country/Region
- Province/State
- Confirmed Cases
- Deaths
- Recovered Cases
- Active Cases

---

## 🎯 Objectives

- Load and explore the COVID-19 dataset.
- Perform data cleaning.
- Analyze daily COVID-19 trends.
- Identify the most affected countries.
- Visualize the data using different charts.
- Generate summary statistics.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

- Data Exploration
- Missing Value Analysis
- Duplicate Removal
- Daily Confirmed Cases Analysis
- Daily Death Cases Analysis
- Daily Recovery Analysis
- Daily Active Cases Analysis
- Top 10 Countries by Confirmed Cases
- Top 10 Countries by Deaths
- Top 10 Countries by Recoveries

---

## 📈 Visualizations

The project includes the following visualizations:

- Line Plot of Daily Confirmed Cases
- Line Plot of Daily Deaths
- Line Plot of Daily Recoveries
- Line Plot of Active Cases
- Bar Chart of Top 10 Countries by Confirmed Cases
- Bar Chart of Top 10 Countries by Deaths
- Bar Chart of Top 10 Countries by Recoveries
- Pie Chart of Top 5 Countries
- Correlation Heatmap
- Histogram
- Box Plot
